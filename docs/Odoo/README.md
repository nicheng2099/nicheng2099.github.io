

# 指令

## anaconda指令

```shell
conda create -n py38 python=3.8
conda init
conda activate py38
```

## 下载依赖

```shell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn
```

## 脚手架

```shell
python odoo-bin scaffold my_name_module myaddons/test
```
