## 添加文章

* 在本文件所在目录创建 markdown 以  `.md` 结尾的文件，比如 `first.md`
* 在 `sidebar.yaml` 中添加侧边栏链接

```markdown
items:
-   label: Brief
    file: README.md
-   label: First
    file: first.md
```
